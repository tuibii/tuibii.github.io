FastDFS分布式文件服务器安装

2019-09-26
  • Linux1
  • 文件服务器1

FastDFS是一款开源的分布式文件系统，功能主要包括：文件存储、文件同步、文件访问（文件上传、文件下载）等，解决了文件大容量存储和高性能访问的问题。FastDFS特别适合以文件为载体的在线服务，如图片、视频、文档等等。

  1. 安装gcc
    yum install make cmake gcc gcc-c++

  2. 下载libfastcommon
    wget https://github.com/happyfish100/libfastcommon/archive/V1.0.39.tar.gz -SO libfastcommon.tar.gz

  3. 解压
    tar -xf libfastcommon.tar.gz

  4. 编译
    ./make.sh

  5. 安装
    ./make.sh install

  6. 下载fastdfs
    wget https://github.com/happyfish100/fastdfs/archive/V5.11.tar.gz -SO fastdfs.tar.gz

  7. 解压
    tar -xf fastdfs.tar.gz

  8. 编译
    ./make.sh

  9. 安装
    ./make.sh install

    程序在 /usr/bin 目录下 配置文件在 /etc/fdfs 目录下
    which fdfs_trackerd
    ls /etc/fdfs

    配置文件是不完整的 把安装包中的配置文件拷贝到 /etc/fdfs 目录下
    sudo cp /usr/local/fastdfs/fastdfs-5.11/conf/* /etc/fdfs

    启动服务
    fdfs_trackerd /etc/fdfs/tracker.conf start
    fdfs_storaged /etc/fdfs/storage.conf start
